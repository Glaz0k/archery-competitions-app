## Модель кубка

### Допустимые типы

```
<cup>
{
  "id": <number>,
  "title": <string>,
  "address": <string | null>,
  "season": <string | null>
}
```

### Пример

```json
{
  "id": 17,
  "title": "Открытый зимний кубок \"Стрелы парадокса\"",
  "address": "г. Санкт-Петербург, ЛК \"Парадокс Лучника\"",
  "season": "2024/2025"
}
```
